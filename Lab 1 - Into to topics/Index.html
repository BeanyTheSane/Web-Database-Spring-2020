What is JavaScript?

JavaScript is a scripting language used mainly for web development in conjunction with HTML and CSS. While HTML provides the content and organization of the website and CSS provides a better visual experience, JavaScript provides interactivity for the user. JavaScript is what is known as a Client-Side script, meaning that it is executed in the clients environment. This allows the client experience to be somewhat dynamic without having the need to request updates from the server and refresh constantly.

What is PHP?

PHP, or PHP: Hypertext Preprocessor, is another scripting language used mainly for web development. Unlike JavaScript however, PHP is a Server-Side scripting language. That means its code is executed on the server. This is also a much more robust tool that's full of features designed to respond to user requests with a personalized html response. This allows for a website to be truly dynamic. Some of the features that make PHP a powerful web development tool are the ability to collect input from a user,interact with different databases, and work with cookies.

What is Ajax?

Ajax stands for Asynchronous JavaScript and XML. Ajax is actually not a programming language but rather, it is a set of principles and techniques for web development. Some of these techniques include HTML/XHTML, the Document Object Model(DOM), XML(plain text or JSON), the XMLHttoRequest oblect, and JavaScript. It works to enhance the user experience by providing speed and functionality. Ajax does this by using JavaScript to make an HTTP request in a seperate thread, interpreting the XML, and updating the interface without reloading the whole page. This results in the user being able to continue to use the current page while the requests are made in the background.
