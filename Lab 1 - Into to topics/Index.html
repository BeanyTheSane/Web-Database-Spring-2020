    • What is ASP.NET Web Forms?

    This was the first programming model available for ASP.NET. The model can be written in a language that supports the Common Language Runtime (CLR) such as visual basic and C#. This model is designed for Rapid Application Development (RAD) and is very designer driven. It allows you to use a drag-and-drop development model to quickly put together the front end views without extensive knowledge of html or javascript. This is possible because the framework will automagically generate most of the markup that will be sent to the browser. This is it's downfall however, because of the auto generated markup you have less control over some of the behaviors of your application.
